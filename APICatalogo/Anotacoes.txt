estrutura de uma api:
    -> \context
        - Aqui fica o ORM
        
    -> \Controllers
        - Aqui fica os controladores da api

    -> \Models
        - Aqui fica os modelos
        
    -> \Repositories
        - Aqui os arquivos de repositorio, que servem para acessar os dados do DB

    -> \Logging
        - Aqui os arquivos para gerar os logs da aplicação
